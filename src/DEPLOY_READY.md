## üêõ TROUBLESHOOTING

### **Issue: Deployment Fails**

**Check:**
1. Vercel Dashboard ‚Üí Deployments ‚Üí Failed deployment
2. View build logs
3. Look for error at bottom

**Common fixes:**
- Clear Vercel cache (Settings ‚Üí General ‚Üí Clear Cache)
- Verify package.json syntax is valid
- Check Node version (should be 18.x)
- Redeploy

### **Issue: Blank White Screen**

**Check:**
1. Browser console (F12) for errors
2. Network tab for 404 errors
3. Verify `/dist/index.html` exists

**Fix:**
```bash
# Test build locally
npm run build
ls -la dist/

# Should show index.html and assets/
# If successful, redeploy to Vercel
```

### **Issue: 404 on Page Refresh**

**Cause:** SPA routing not configured

**Fix:** Verify `vercel.json` has rewrites:
```json
{
  "rewrites": [
    { "source": "/(.*)", "destination": "/index.html" }
  ]
}
```

**Already configured in your project! ‚úÖ**

### **Issue: Features Not Working**

**Check:**
1. Console for JavaScript errors
2. localStorage available? (console: `typeof localStorage`)
3. Try different browser

**Fix:**
- Clear browser cache
- Try incognito mode
- Check for browser extensions blocking
